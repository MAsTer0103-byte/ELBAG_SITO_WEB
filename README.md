# ELBAG Modern UI

Sito moderno e minimalista per ELBAG SIMP realizzato con Next.js, Tailwind CSS,
shadcn/ui e Framer Motion.

## Script principali

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Struttura

- `src/app/page.tsx` compone le sezioni principali.
- `src/components/sections` contiene i blocchi riutilizzabili.
- `src/components/ui` ospita i componenti shadcn/ui.
